# Telegram Bot Verification System

A production-ready verification system for Telegram bots with Linkvertise monetization and MongoDB Atlas storage.

## 🚀 Features

- Secure token generation using crypto.randomBytes()
- MongoDB Atlas integration with automatic TTL cleanup
- Linkvertise Full Script API integration
- Modern, responsive verification page
- Rate limiting and security headers
- Automatic token expiration after 15 minutes
- One-time use tokens
- Production-ready Vercel deployment

## 📋 Prerequisites

- Node.js 18+
- MongoDB Atlas account
- Vercel account
- Telegram Bot
